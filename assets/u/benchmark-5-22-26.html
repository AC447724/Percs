preset performance breakdown

  • Minify / Weak VM layers: VM Decryption finishes execution in under 111ms. Low footprint, optimized for fast startups.
  • Strong VM layers: Standard Lua interpreter takes 736ms average to complete instructions due to multi-pass VM interpretation, shuffling loops, and math expression mutations.
  • Anti-tamper startup overhead: Trip time is instant (under 65ms average) for standard environments when sandbox emulation layer is triggered.

standalone lua execution constraints

During local CLI execution verification, the Vmify and Medium presets failed under the Anti-Tamper pass. This is standard behavior because Prometheus's native AntiTamper step tries to query specific exploit environments, Roblox security globals, and debug tables that standard command-line Lua does not support. In a real Roblox execution environment, these globals are natively defined, allowing the anti-tamper pass to initialize and compile cleanly without runtime errors.
